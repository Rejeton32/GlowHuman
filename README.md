# GlowHuman

## Project Overview
GlowHuman is an innovative project designed to improve human well-being through the use of technology. It leverages modern development frameworks and libraries to provide a seamless experience for users.

## Tech Stack
- **Frontend:** React, Redux, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Deployment:** Docker, AWS

## Setup Instructions
1. Clone the repository:  `git clone https://github.com/Rejeton32/GlowHuman.git`
2. Navigate to the `client` and `server` directories and install dependencies:
   - `cd client && npm install`
   - `cd ../server && npm install`
3. Set up environment variables in a `.env` file.
4. Start the server: `cd server && npm start`
5. Run the client in another terminal: `cd client && npm start`
